Introduction

Welcome to RetroLauncher! This tool allows you to manage and launch emulators and games with ease.

Configuration

  • Add emulators by providing their executable paths and supported consoles.
  • Add ROMs with associated consoles, genres, and favorite status.
  • Set global and per-game launch parameters.

Usage

  1. To launch a game, find it in the main table.
  2. If a ROM supports one emulator, it launches immediately upon selection.
  3. If multiple emulators are supported, a selection window will appear.

Launch Parameters

  • {emu}: Specifies the position of emulator parameters.
  • {rom}: Specifies the position of the game file path.
  • Example: --fullscreen {emu} {rom}

Troubleshooting

  • Ensure emulator and ROM paths are correct.
  • Check if required files are missing.
  • Contact support if issues persist.

Frequently Asked Questions

How do I add a new emulator?

Go to "Manage > Emulators" and fill in the required details.

Can I adjust game launch parameters?

Yes, parameters can be customized per game and emulator.
